# ClimateServ API Access

This is a simple package to access the [ClimateSERV API](https://climateserv.servirglobal.net/) 

## Current supported operations:
* Timeseries
    * Average
    * Min
    * Max
* Download tifs

## Current supported datasets:
* CHIRPS
    * Rainfall
    * GEFS
        * Anomalies
        * Precipitation 
            * Mean
            * 25th percentile
            * 75th percentile
* eMODIS
    * Central Asia NDVI
    * East Africa NDVI
    * Southern Africa NDVI
    * West Africa NDVI
* Seasonal_Forecast
* IMERG 1 Day
* Evaporative Stress Index
    * ESI 4 week
    * ESI 12 week
